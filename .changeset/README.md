# Changesets

This public repo covers the first-wave Xapps JavaScript SDK packages:

- `@xapps/server-sdk`
- `@xapps/widget-sdk`
- `@xapps/signing-client`
- `@xapps/openapi-import`
- `xapps-embed-sdk` (public release name: `@xapps/embed-sdk`)
- `@xapps/browser-host`
- `@xapps/backend-kit`

Out of scope:

- `@xapps/cli`
- app workspaces
- sample/demo workspaces

Use Changesets here for version intent and coordinated public release notes.
